# Code Of Conduct

Be respectful, direct, and constructive.

This project welcomes practical contributions that improve the CRM while keeping
users, maintainers, and customer data safe. Harassment, abuse, personal attacks,
credential sharing, or public disclosure of sensitive security details are not
acceptable.

Maintainers may edit, hide, or remove comments, issues, pull requests, and
contributions that violate this standard or put users at risk.
